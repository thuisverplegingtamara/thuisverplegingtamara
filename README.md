# Tamara Thuisverpleging

Publieke website voor **Tamara Thuisverpleging**.

Website: **https://www.thuisverplegingtamara.be**

Tamara Thuisverpleging biedt zorg aan huis in de regio **Zonhoven, Houthalen en Zolder**, met aandacht voor algemene verpleging en wondzorg.

## Over deze repository

Deze repository bevat uitsluitend bestanden die nodig zijn om de publieke website te publiceren.

Omdat deze repository publiek zichtbaar is, bevat deze README bewust geen technische implementatiedetails, backendconfiguratie, database-informatie, sleutelbeheer, beheerprocedures, testinstructies, interne operationele documentatie of informatie over beveiligde zones.

## Privacy en veiligheid

Plaats nooit de volgende informatie in deze repository, issues, pull requests, commits, screenshots of bijlagen:

- patiëntgegevens;
- medische informatie;
- documenten uit patiëntendossiers;
- toegangscodes, tokens, secrets, private keys of API-sleutels;
- database-exports, back-ups of logs;
- interne beheerprocedures;
- screenshots van beheerschermen of beveiligde zones;
- technische details die misbruik kunnen vergemakkelijken.

Gevoelige gegevens moeten uitsluitend beheerd worden via daarvoor voorziene, beveiligde systemen.

## Veiligheidsproblemen melden

Meld vermoedelijke veiligheidsproblemen niet via publieke GitHub issues.

Gebruik de contactgegevens op de website en deel alleen de minimaal noodzakelijke informatie om het probleem te kunnen opvolgen. Deel geen patiëntgegevens, tokens, logs, screenshots met gevoelige inhoud of technische exploitdetails via publieke kanalen.

## Medische urgenties

Deze website en deze repository zijn niet bedoeld voor dringende medische communicatie. Neem bij medische urgentie contact op met de bevoegde hulpdiensten.

## Rechten

Alle teksten, logo's, beelden, huisstijlelementen en websitebestanden zijn eigendom van de rechthebbenden. Hergebruik of verspreiding zonder toestemming is niet toegestaan.
